# Roadmap

The roadmap is intentionally small. Dates are not promised; reliability and
privacy take priority over feature count.

## 1.0.x - Reliability

- Improve first-run permission guidance and unsigned-app installation help.
- Expand tests around capture exclusions, deletion, export, and database upgrades.
- Fix issues found by early public users.

## 1.1 - Everyday use

- Refine capture feedback and privacy status.
- Improve board, tag, and bulk-organisation workflows.
- Make local-AI setup and offline states easier to understand.

## Later

- Evaluate import and backup workflows that preserve local ownership.
- Consider update checks only if they can remain transparent and opt-in.

Ideas that require cloud accounts, telemetry, or uploading captures are outside
Hort's current direction.
